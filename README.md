# 🩸 Rakta-Vahini — Blood Donor App

An AI-powered Android app to connect blood donors with people in 
need during medical emergencies.

## Features
- 🔍 Search for blood donors by blood group and location
- 🚨 Send emergency blood requests instantly
- 📞 Contact donors directly via call or message
- 🔥 Real-time data powered by Firebase

## Tech Stack
- Android (Kotlin + Jetpack Compose)
- Google AI Studio (Generative AI)
- Firebase (Realtime Database + Auth)
- Capacitor Framework
